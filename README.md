# fundamental-regime-report

**기업 펀더멘털 + 시장 국면 인지형 통합 분석 리포트 시스템**

한국 주식시장(KOSPI 중심) 종목에 대해

1. **펀더멘털**: 재무제표 기반 건전성·부실 위험 스코어링 + LLM 서술형 리포트
2. **시장 국면(Market Regime)**: 지수 수익률·변동성 기반 국면 *분류* (예측 아님)
3. **통합 대시보드**: 선택 기업의 주가·국면·펀더멘털 리포트를 한 화면에서,
   리스크 해석을 **국면에 조건부**로 제시

를 제공하는 배치 기반 분석 시스템입니다.

> **포지셔닝**: 1인 개발자의 포트폴리오 프로젝트.
> *방법론적 엄밀성*(데이터 누수·룩어헤드 차단, 시간 인지 분할)과
> *명확한 스코프*(한국·배치·고정 유니버스·예측 아님)를
> 모든 기술적 욕심보다 우선합니다.
